### 基本用法

``` html
 <div>
    <cw-antd-affix :offset-top="10">
      <a-button type="primary" @click="10 + 10">
        Affix top
      </a-button>
    </cw-antd-affix>
    <br />
    <cw-antd-affix :offset-bottom="bottom">
      <a-button type="primary" @click="bottom += 10">
        Affix bottom
      </a-button>
    </cw-antd-affix>
  </div>
```
